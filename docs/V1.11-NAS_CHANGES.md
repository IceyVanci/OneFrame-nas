## 二、Type L 导出模糊背景缩放修复

修复 Type L 样式导出图片的模糊背景边缘黑边/透明边问题。

**根因：** 预览端 (`type-L.css`) 使用 CSS `transform: scale(1.5)` 将模糊背景从中心放大 1.5 倍，隐藏 blur 产生的透明边缘。但导出端 (`type-L-export.js`) 的 `drawBlurredBackground` 函数只做了 cover 裁剪，没有额外缩放。

**修复：** 在 `drawBlurredBackground` 中添加 1.5x 缩放，与预览端 CSS `transform: scale(1.5)` 一致。

### 导出端注意事项

导出端文字位置必须基于照片区域（photoX/photoY/photoW/photoH）计算，而非画布尺寸（canvasWidth/canvasHeight）。底部锚点为 `photoY + photoH * 0.97`，X 坐标基于 `photoX + photoW * 比例`，确保文字在清晰照片区域内。

## 四、样式 B/F/G/L 图片圆角

为样式 B、F、G、L、M 的图片添加 12px 圆角，预览端和导出端效果一致。

### 预览端（CSS）

添加 `border-radius: 12px`：

| 文件 | 选择器 | 说明 |
|------|--------|------|
| `css/type-B.css` | `.frame-wrapper.type-b > img` | Type B 图片圆角 |
| `css/type-F.css` | `.frame-wrapper.type-f img` | Type F 图片圆角 |
| `css/type-G.css` | `.frame-wrapper.type-g > img` | Type G 图片圆角 |
| `css/type-L.css` | `.frame-wrapper.type-l > img:not(.type-l-blur-bg)` | Type L 图片圆角（排除模糊背景层） |
| `css/type-M.css` | `.frame-wrapper.type-m > img:not(.type-m-blur-bg)` | Type M 图片圆角（排除模糊背景层） |

### 导出端（Canvas）

使用 `ctx.roundRect()` + `ctx.clip()` 裁剪，圆角半径按画布宽度比例缩放：

```javascript
const baseScale = canvasWidth / 900;
const cornerRadius = Math.round(12 * baseScale);
ctx.save();
ctx.beginPath();
ctx.roundRect(photoX, photoY, photoWidth, photoHeight, cornerRadius);
ctx.clip();
ctx.drawImage(img, ...);
ctx.restore();
```

**缩放说明**：预览基准宽度 900px 对应 12px 圆角，导出时按实际画布宽度等比缩放（如 6000px 宽 → 80px 圆角），确保视觉比例一致。
