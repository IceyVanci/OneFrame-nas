## 一、Type J 编辑面板改进

### 设备型号输入框显示

Type J 编辑面板现在显示设备型号输入框，方便用户手动修改机型名称。

**变更：**
- 删除 `type-j-editor-panel.js` 中隐藏设备型号区域的代码
- 输入框 placeholder 改为"型号（如 Sony A7M4）"
- 机型名称仍默认从 EXIF 读取含厂商前缀（与 Type F 相同逻辑）

## 二、Type K 左下角 Logo + 双行文字样式（移植自原始项目 v1.0.8）

### 设计说明

新增 Type K 样式（左下角 Logo + 双行文字），基于 Type H/I/J 的代码结构实现。

**布局：**
- Logo 在底部左下角（5% 起）
- Logo 右侧第一行：署名（medium）+ 日期（normal）
- Logo 右侧第二行：机型名称（medium）+ 拍摄参数（normal）
- 机型名称不含厂商前缀

**与 Type H 的关键差异：**

| 特性 | Type H | Type K |
|------|--------|--------|
| Logo 位置 | 底部居中 | 左下角 |
| 文字布局 | 三行居中 | 两行左对齐（Logo 右侧） |
| 字重 | 全部 normal | 署名/机型 medium，其余 normal |
| 机型名称 | 不含厂商 | 不含厂商 |

|------|------|
| `src/renderer/css/type-k.css` | Type K 样式定义 |

## 三、Type L 高斯模糊背景样式（移植自原始项目 v1.0.8）

### 设计说明

新增 Type L 样式（高斯模糊背景），基于 Type G 修改，将白色外框替换为照片的高斯模糊版本。

**与 Type G 的关键差异：**

| 特性 | Type G | Type L |
|------|--------|--------|
| 外框背景 | 白色 #ffffff | 照片高斯模糊 |
| 文字颜色 | 黑色 #000000 | 白色 #ffffff（默认） |
| 照片区域 | 相同（92%×80%） | 相同 |
| 编辑面板 | 显示 Logo + 署名 | 同 Type G + 文字颜色选择 |

### 新建文件

| 文件 | 说明 |
|------|------|
| `src/renderer/css/type-l.css` | Type L 样式定义（blur 背景层） |
| `src/renderer/js/styles/type-l-preview.js` | Type L 预览模块（动态创建 blur DOM） |
| `src/renderer/js/styles/type-l-export.js` | Type L 导出模块（Canvas 两步绘制） |
| `src/renderer/js/components/type-l-editor-panel.js` | Type L 编辑面板配置 |

### 技术细节

**预览端：**
- 动态创建 `<img class="type-l-blur-bg">` 作为模糊背景层
- CSS `filter: blur()` + `transform: scale(1.5)` 解决 blur 透明边缘问题
- `overflow: hidden` + `background: #000` 裁剪溢出区域

**导出端：**
- Canvas `ctx.filter = 'blur()'` 两步绘制
- 第一步：全画幅模糊背景（cover 裁剪）
- 第二步：清晰照片 92%×80% 居中
- 第三步：绘制白色文字内容
