# OneFrame NAS v1.03 移植说明

**日期**：2026-06-18

---

## 概述

同步原始项目 v1.0.3 的 Type F 画中画样式到 NAS 版本。

## 新增 Type F 边框样式

### 布局设计
- 画布比例 = 图片比例（宽度 = 图片宽度，高度 = 图片高度 / 0.8）
- 顶部 5% 白色留白 + 中部 92%×80% 照片展示区 + 底部 15% 文字信息区
- 文字区域使用绝对定位，署名不影响前两行位置
- 字号动态缩放（基准 900px 宽度对应 14px）

### 文字内容
- 第一行："Shot on"（灰色 #888）+ 品牌型号（黑色 #000）
- 第二行：日期 光圈 焦距 快门 ISO（全部灰色 #888）
- 第三行：© 署名（灰色 #888，可选）

### 编辑面板
- 隐藏边框颜色、边框高度、比例设置、Logo 区域
- 设备型号文本框自动包含品牌名（如 "Sony A7M4"）
