# OneFrame NAS v1.0.0 移植说明

**日期**：2026-06-16

---

## 概述

将 OneFrame 从 Electron 桌面应用移植为纯前端 Web 应用，通过 Docker 容器化部署在 NAS 上。

## 架构变更

- 移除 Electron 28.0，采用 Nginx + Docker 方案
- 零构建方案：原生 HTML/CSS/JS + nginx:alpine 静态服务器
- 浏览器原生 API 替代 Electron API（`<input type="file">`、`<a download>`、`URL.createObjectURL()`）
